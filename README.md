### プロダクト概要
Scaffoldはユーザーが日常で抱く些細な疑問を深く理解する踏み台となります。些細な疑問はその個々人の趣味趣向を反映しており、それらを追求することは不確実性の時代において固有の価値を提供するための強固な土台になると感じています。

ユーザーは事前にどのような目標があるか、現状はどのようなスキルセットかなどを整理し、日常の中で気になったトピックをAIアシスタントに伝えることで、あなたと共に最適な学び方を対話形式で計画します。AIとユーザーが共に納得のいく最適な学習方針が出来上がった時、Scaffoldエージェントは自動でロードマップから学習教材の作成、質問回答まで伴走します。

学習進捗は単純なチェックと、どのくらい深く読み込んだかの二つを入力することで、全てをはじめから完璧にするという心理障壁を抑え、自身の興味を中心に前に進み、興味が湧いた時に手薄な部分に戻るといったスタイルを促します。
